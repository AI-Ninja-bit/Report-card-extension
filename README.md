# 📄 Report Card Generator Chrome Extension

Generate clean PDF report cards directly from school websites.

## ✨ Features
- Auto extracts student data
- Generates formatted PDF
- Works with Hindi + English layouts
- One-click download

## 📦 Installation (Manual)
1. Clone this repo
2. Go to chrome://extensions
3. Enable Developer Mode
4. Click "Load Unpacked"
5. Select this folder
